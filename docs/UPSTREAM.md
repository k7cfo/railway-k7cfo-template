# Upstream provenance

Portable SaaS Template began from `knowsuchagency/cloudflare-template`, cloned on 2026-07-16 from its `main` branch at commit `4f7b63d` (the shallow clone's imported state).

Preserved assets include the Nothing, Apex, and Onyx CSS design systems, font selections, design tokens, marketing motion components, and shadcn/Radix UI foundations. The runtime was changed from Cloudflare Workers/D1/Bun to a portable Node/Hono/PostgreSQL/pnpm container.

No standalone `LICENSE` file was present in the cloned upstream repository. This project does not invent or replace an upstream license; upstream attribution and notices are retained here and in the root README.
